Chapter 1.
1. Come up with : (돈 해결책 아이디어 등을) 생각해내다 , 마련하다
=> We need to come up with something fast. : 해결책을 빠르게 마련해야한다.
2. put together : (이것 저것 모아) 만들다. 준비하다.
=> Could you put together a presentation for tomorrow. : 내일 프레젠테이션 만들어 줄 수 있어?
3. work : 1) 일하다 / 2) (일정이) 되다 / 3. 작동하다
=> I have to work late tonight : 늦게 일해야만해
=> Friday doesn’t work : 금요일 안 된다.
=> I don’t think the wifi is working
4. keep 사람 posted : (진행 상황/현황을) 계속해서 ~에게 알려주다.
=> keep me posted : 내게 계속 알려줘
5. move : 일정을 조정하다
=> Can you move the meeting up? : 회의 일정을 더 일찍 당길 수 있어?
6. pick up : 1) (중단된 상황부터) 계속하다 /  (2) 사다
=> Let’s pick this up tomorrow. : 내일 마저 이야기하자
=> I’ll go pick up some coffee : 커피 사올게!
3) pick up on : 눈치채다
=> Did you pick up on that?
7. let 사람 know : ~에게 말해줘, 알려줘
=> Let me know if anything changes. : 뭐 바뀌면 알려줘
8. at the latest : 늦어도
=> Friday, at the latest : 늦어도 금요일
9. figure out : (곰곰이 생각한 후) 알아내다. / 이해하다
=> I’ll figure out something : 해결책을 마련해볼게.
10. ballpark figure : (어림잡은 숫자)
=> Give me a ballpark figure, how much? : 어림 잡아서 얼마?
11. be stuck : (안 좋은 상황에, 장소에 갇혀) 꼼짝 못하다
=> I don’t mean to be late, but I’m stuck in traffic : 늦으려고 한 건 아닌데, 차가 막혀 꼼짝 못하고 있어
12. I want to say : (진심을 담아) ~라고 꼭 말하고 싶어
=> I wanna say thank you : 진심 땡큐
13. I mean it : 진심이야
=> You are the best. I mean it : 너가 최고야 진심.
14. grab : 급히, 잠깐 ~하다
=> Maybe we can grab up a cup of coffee sometime? : 아마 우리 언젠가 커피 마실수 있겠지?
15. company : 동료/ 같이 있어줌
=> I really enjoyed your company the other night. : 저번 밤 너와의 시간이 즐거웠어.
16. work : 직장
=> I’m still at work. : 나 아직 회사야
17. Is this a good time? : 통화하기 괜찮아?
18. Did I catch you at a bad time? : (상대가 바빠 보일 때) 통화하기 힘들어?
19. Sorry to call so late. : 너무 늦은 시간에 전화해서 미안
20. Sorry to call so early. : 이른 시간에 전화해서 미안
21. May I ask who’s calling? : 통화하시는 분 성함 여쭤봐도 될까요?
22. May I ask what this is regarding? : 용건이 뭔지 여쭤봐도 될까요? / 뭐 때문에 그러시죠?
23. To what do I owe this pleasure? : (기쁜마음을 담아 도와주고 싶은 뉘양스)
= How can I help you? ( 나이 든 사람이 쓰는 표현)
24. at the moment. : 지금 이순간
=> I’m tied up(swamped, busy) at the moment. : 제가 지금 바빠요.
25. When can I expect to hear from you? : 언제쯤 연락주실건가요?
26. be in touch
=> We will be in touch with the figures. : 조만간 연락드릴게요
27. ASAP : as soon as possible 가능한 빨리
28. hesitate : 주저하다
=> If there’s anything you need, don’t hesitate to call me : 필요한게 있다면 언제든 연락하세요.
29. feel free to + V
=> If you have any questions, feel free to ask. : 질문있으면 언제든 질문해
30. It was nice to talking to you. : (전화 끊기 전) 통화해서/대화해서 즐거웠어
31. You’re the best : 정말 고마워
32. catch up: 못 다한 얘기를 다하다
=> We should catch up sometime. : 언제 만나서 못 다한 이야기를 하자
33. get together : 모이다
=> What do you say(어떻게 생각해?) we get together for a drink? : 만나서 술마실까?
34. find : 찾다, 발견하다 / 여기다, 생각하다
=> I find it very challenging : (어렵지만 도전해볼만한 긍정적인 느낌) 난 그게 어렵다고 생각해
35. doable : 할만하다
=> It’s challenging but doable : 어렵지만 할만해
36. at all : (의무문/조건문에서 사용) 조금이라도
=> Did you sleep at all last night? : 어젯밤 조금이라도 잤어?
=> If you need anything at all, please just give us a ring : 필요한거 조금이라도 있으면 전화주세요
37. be in the middle of + 명사 : ~하느라 바쁘다. (~하는 중이다)
=> I’m in the middle of something here. 여기서 뭐좀 하느라 바빠.
38. I didn’t mean to be late : 늦으려는 건 아니였어
39. How did it go? : 어떻게 진행됐어?
40. Whatever it is, it’s gonna have to wait. : 뭐하고 있었든 잠깐 기다려봐
